## 二、沪日 IX 专线评测

---

### 0、什么是 IX 专线？

> 以下内容参考自 [《IX 专线全解：基于云内网接入的全球安全互联方案》](https://www.mkcloud.net/index.php/knowledgebase/36/IX%E4%B8%93%E7%BA%BF%E6%98%AF%E4%BB%80%E4%B9%88%E4%B8%80%E6%96%87%E5%B8%A6%E4%BD%A0%E4%BA%86%E8%A7%A3%E4%BA%92%E8%81%94%E7%BD%91%E4%BA%A4%E6%8D%A2%E4%B8%93%E7%BA%BF.html)

**IX 专线**，即互联网交换中心专线（Internet Exchange专线），其核心底层逻辑是**基于大型云供应商生态的定向互联**：

- **强制云厂进站**：IX 专线必须通过阿里云、腾讯云、华为云等大型厂商的内网环境进行对接，入口地址隐藏在云厂内网中，公网不可见。
- **私有专线中转**：流量进入云厂内网后，通过专用私有物理链路进行中转传输，绕过拥堵的公网海缆。
- **全球公网出口**：流量到达境外 IXP（互联网交换中心）后，不仅可以与大厂路由对等，更可以通达全球公网，满足全网访问需求。

相比传统 IEPL/IPLC 专线，IX 专线具有**物理级隐身（抗 DDoS）、低延迟高稳定、整体成本可降低 30%-50%** 的优势，尤其适合已托管在云端的企业进行跨境组网。

## 3、iperf3 网络测速

### 3.1 华北移动方向

#### TCP 测试

```
# Client (华北移动)
[  5] local 172.16.1.38 port 3899 connected to 172.16.1.3 port 8505
[ ID] Interval           Transfer     Bitrate         Retr
[  5]   0.00-15.03  sec   154 MBytes  86.0 Mbits/sec    0            sender

```

#### UDP 测试

```
# Client (华北移动)
$ iperf3 -c 211.136.*.* -p 3899 -P 1 -R -u -b 88M
[  5]   0.00-10.00  sec   102 MBytes  85.2 Mbits/sec  0.200 ms  136/74755 (0.18%)  receiver
```

**华北移动小结**：TCP/UDP 均能稳定跑满 **85~86 Mbps**，零重传，UDP 丢包率仅 0.18%，线路非常稳定。不过各地区运营商有可能有针对UDP的QoS策略，有可能会遇到本地运营商的限速或阻断，这是没有办法的事情。

#### UDP 去程测试

```
$ iperf3 -c 114.111.*.* -p 3899 -P 1 -t 15 -u -b 88M
[  5]   0.00-15.00  sec   157 MBytes  88.0 Mbits/sec  0.000 ms  0/113951 (0%)  sender
[  5]   0.00-15.01  sec  0.00 Bytes  0.00 bits/sec  0.000 ms  0/0 (0%)  receiver
```

#### UDP 回程测试

```
# Client (阿里云上海)
$ iperf3 -c 114.111.*.* -p 3899 -P 1 -t 15 -u -b 88M -R
[  5]   0.00-15.00  sec   153 MBytes  85.5 Mbits/sec  0.269 ms  33/110740 (0.03%)  receiver

#### TCP 回程测试

```
# Client (阿里云上海)

**上海阿里云小结**：
- UDP 去程可跑满 88 Mbps（零丢包）
- UDP 回程稳定在 **85.5 Mbps**，丢包率仅 0.03%
- TCP 回程稳定在 **82.9 Mbps**
- 阿里云上海作为前置入口表现也不错，不过上海阿里云到IX入口还有9ms的延迟，实际体验稍差于商家的移动入口。

## 5、简单评价

- **产品定位**：NoBrand 沪日 IX 专线主打低延迟、高稳定的沪日企业组网/跨境加速场景，非普通"翻墙"用途，更适合有合规跨境组网需求的用户。
- **网络表现**：
  - 端内延迟稳定在 **28 ms** 左右，抖动极低（< 0.4 ms），丢包率接近 0，属于非常优质的专线水准。
  - TCP/UDP 双向均能稳定跑满 **85 Mbps** 左右带宽，重传率为 0，线路质量可靠，更高套餐有更高的带宽。
  - 阿里云上海实测可作为优质前置入口，其他公有云建议自行测试。
- **合规与限制**：
  - 移动 IX 合体版本**强制实名认证**，但省却了自行购买前置入口的麻烦。
  - **协议限制严格**：不支持 SS/VLESS/VMESS/Trojan 等翻墙协议，仅支持 WireGuard/OpenVPN/IPsec/AnyConnect 等企业组网协议，购买前务必确认自身业务协议合规。
  - 联通用户需注意**山东、河南、北京**不可用；电信用户需核对所在省份是否在可用列表内，没在表内的不清楚是否可可用，请自测。
- **618 活动**：移动 IX 合体版本和纯沪日 IX 专线均参与 8.5 折，叠加限量 66 折优惠券后性价比不错。
- **购买建议**：
  - 如果你需要**合规的企业组网/跨境加速**，且所在地区在可用列表内，这条专线值得考虑。
  - 如果你需要"翻墙"协议，请自测没有列出的协议，或者选择纯沪日 IX 自备前置入口方案。
  - 建议月付起步，先测试本地网络可用性再决定是否长期持有，后续可以选择工单补差价升级到年付。
